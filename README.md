# Cesar Naim - Obsidian Metamorphosis

Sitio web oficial y plataforma de mentoría exclusiva para **Cesar Naim**. Este proyecto encarna la filosofía de **"Obsidian Metamorphosis"**, un espacio de transformación de alto rendimiento diseñado para personas de alto perfil, combinando rigor científico con profundidad filosófica.

El diseño del sitio sigue una estética de **Minimalismo Sofisticado** con una fuerte influencia de **Tactile Noir**, utilizando un fondo negro absoluto (`#000000`) y acentos de color de alta gama (Azul Cobalto, Vinotinto Profundo y Oro Metálico) para evocar una sensación de exclusividad, misterio y transformación.

---

## 🛠️ Tecnologías Utilizadas

La arquitectura del proyecto está construida sobre las siguientes tecnologías y herramientas modernas:

- **Framework principal**: [Next.js 15](https://nextjs.org/) (App Router)
- **Biblioteca de interfaz**: [React 19](https://react.dev/)
- **Lenguaje**: [TypeScript](https://www.typescriptlang.org/)
- **Estilos**: [Tailwind CSS v4](https://tailwindcss.com/) (PostCSS integration)
- **Componentes base**: Radix UI
- **Iconografía**: [Lucide React](https://lucide.dev/)
- **Gestión de paquetes**: [pnpm](https://pnpm.io/)

---

## 🎨 Sistema de Diseño (Obsidian Metamorphosis)

- **Colores de Acento**:
  - **Primario (Azul Cobalto):** Utilizado para marcadores científicos y subencabezados.
  - **Secundario (Vinotinto Profundo):** Reservado para secciones temáticas emocionales y de cableado neuronal.
  - **Acento de Lujo (Oro Metálico):** Utilizado para llamadas a la acción (CTAs) principales y divisores estructurales.
- **Tipografía**:
  - **Playfair Display**: Para encabezados con un toque editorial y de alta costura.
  - **Hanken Grotesk**: Para textos de lectura y etiquetas, con pesos ligeros y espaciado amplio.
- **Bordes y Sombras**: Estructuras geométricas angulares puras (sin bordes redondeados, `rounded-none`) y profundidad creada a través de bordes finos de 1px y gradientes radiales de luz difusa.

## 🚀 Inicio Rápido (Desarrollo Local)

Sigue estos pasos para ejecutar el proyecto en tu entorno local:

### 1. Clonar el repositorio e instalar dependencias
Asegúrate de tener instalado [pnpm](https://pnpm.io/). Luego, instala las dependencias del proyecto:

```bash
pnpm install
```

### 2. Ejecutar el servidor de desarrollo
Inicia el entorno de desarrollo local con:

```bash
pnpm dev
```

El sitio estará disponible en [http://localhost:3000](http://localhost:3000).

### 3. Compilación de producción
Para construir la aplicación optimizada para producción:

```bash
pnpm build
pnpm start
```
